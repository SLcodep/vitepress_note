## node的配置使用
首先，确保你已经安装了Node.js。然后，创建一个新的项目，并安装Express
```yaml
mkdir my-mock-server
cd my-mock-server
npm init -y
npm install express
```
下面通过简单的node+express模拟后端接口


::: code-group
```js [get方法]
const express = require('express');
const cors = require('cors');   //配置跨域
const app = express();
const PORT = 3000;


// 解析JSON格式的请求体（如前端POST JSON数据）
app.use(express.json());
app.use(cors()); // 使用cors中间件，允许跨域请求
// 解析URL编码的请求体（如表单提交）
app.use(express.urlencoded({ extended: true }));

const students = [
  { name: '张三', studentId: '2024001', height: 175 },
  { name: '李四', studentId: '2024002', height: 180 },
  { name: '王五', studentId: '2024003', height: 172 },
  { name: '赵六', studentId: '2024004', height: 168 }
];

// 前端请求示例：http://localhost:3000/api/get-data?name=张三&age=20
app.get('/api/get-data', (req, res) => {
  // 构造返回数据：包含请求的参数、请求头、请求方法等
  const responseData = {
    code: 200, // 状态码
    msg: 'GET请求成功', // 提示信息
    requestInfo: {
      method: req.method, // 请求方法（GET）
      url: req.originalUrl, // 完整请求URL
      query: req.query, // GET参数（{ name: '张三', age: '20' }）
      headers: req.headers, // 请求头（如origin、user-agent等）
      ip: req.ip // 客户端IP
    },
    data:students 
  };
  // 返回JSON格式数据给前端
  res.json(responseData);
});

```
```js [post方法]
// 前端POST示例：请求地址 http://localhost:3000/api/post-data，请求体 { "username": "admin", "password": "123456" }
app.post('/api/post-data', (req, res) => {
  const responseData = {
    code: 200,
    msg: 'POST请求成功',
    requestInfo: {
      method: req.method, // 请求方法（POST）
      url: req.originalUrl,
      body: req.body, // POST请求体（{ username: 'admin', password: '123456' }）
      query: req.query, // 若URL带参数也能获取（如/api/post-data?token=123）
      headers: req.headers
    }
  };
  res.json(responseData);
});
```
:::

启动的方式是通过node命令
```js
//server.js
node server.js
```

:::tip 
上面只是简单的实现一个接口，node作为前端走向全栈的一个重要技术。
- **node的学习路线**
  - **文件系统模块(fs)**：文件的读写(异步、同步)
  - **HTTP**:`request`对象封装了HTTP请求,`response`对象封装了HTTP响应
  - **path模块**：用于处理和转换文件路径，这个API提供了多个方法来操作字符串形式的路径
- **node的进阶**
  - **koa**：koa是Express的下一代基于Node.js的web框架
  - **数据库**：操作数据库
:::
