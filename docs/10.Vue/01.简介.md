## Vue的学习流程
- 1.设计哲学
- 2.响应式原理
    - ref、retivate
    - 响应式系统
- 3.核心API-vue的核心能力
    - 响应式
    - 生命周期
    - 组件化
    - 组件通信
    - 状态管理
- ​4.工程化

| **Vue3** | 响应式基础（组合式） | ref、reactive、computed、watch、watchEffect                  | 定义响应式数据（基础 / 对象类型）、计算值、监听（显式 / 自动收集依赖） |
| -------- | -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
|          | 响应式工具           | readonly、toRef、toRefs、shallowRef、shallowReactive         | 只读数据、响应式属性解构、浅响应式（性能优化）               |
|          | 生命周期（组合式）   | onMounted、onUpdated、onUnmounted、onErrorCaptured           | 组件生命周期钩子（需导入使用，对应 Vue2 同名钩子）           |
|          | 组件与上下文         | setup、defineProps、defineEmits、defineExpose、useAttrs、useSlots | 组合式 API 入口、定义 props / 事件、暴露组件内部、获取属性 / 插槽 |
|          | 全局 API（模块化）   | createApp、defineComponent、h(createVNode)、provide/inject   | 创建应用实例、定义组件（TS 友好）、创建虚拟 DOM、跨组件通信  |
