## 浏览器输入URL后发生了什么？
> 参考内容-[在浏览器上输入一个URL后发生了什么？](https://blog.csdn.net/lichunericli/article/details/140582342?ops_request_misc=%257B%2522request%255Fid%2522%253A%25222369273ee92b30ab868196600c401e26%2522%252C%2522scm%2522%253A%252220140713.130102334.pc%255Fall.%2522%257D&request_id=2369273ee92b30ab868196600c401e26&biz_id=0&utm_medium=distribute.pc_search_result.none-task-blog-2~all~first_rank_ecpm_v1~rank_v31_ecpm-1-140582342-null-null.142^v102^control&utm_term=%E6%B5%8F%E8%A7%88%E5%99%A8%E8%BE%93%E5%85%A5URL%E5%90%8E%E5%8F%91%E7%94%9F%E4%BA%86%E4%BB%80%E4%B9%88%EF%BC%8Chttp%E7%BC%93%E5%AD%98%E5%8F%91%E7%94%9F%E5%9C%A8%E5%93%AA%E4%B8%AA%E9%98%B6%E6%AE%B5&spm=1018.2226.3001.4187)

1.URL解析，判断输入的是否是合法的URL
2.对网址进行DNS域名解析，得到对应的IP地址
3.根据这个IP，找到对应的服务器，发起TCP的三次握手
4.建立TCP连接后，发起HTTP请求
5.服务器响应HTTP请求，浏览器得到html代码
6．浏览器解析 html 代码，并请求 html 代码中的资源（如 js、css、图片等）（先得到 html 代码，才能去找这些资源）
7.浏览器对页面进行渲染呈现给用户
8.服务过程完毕，关闭 TCP 连接，四次挥手
