## 📋1. Frontmatter 配置说明

```yaml
---
title: 文章标题                    # 必填，页面标题
date: 2025-11-27 17:24:35         # 必填，创建日期（不会自动更新）
permalink: /category/article      # 必填，永久链接
titleTag: 原创                    # 可选，标题右侧标签
categories:                       # 可选，文章分类
  - 前端
  - Vue
tags:                            # 可选，文章标签
  - JavaScript
  - TypeScript
author:                          # 可选，覆盖全局作者
  name: 作者名
  link: https://github.com/xxx
sticky: 1                        # 可选，置顶（数字越大越靠前）
---
```

```js
function hello() {
  console.log('旧代码')  // [!code --]
  console.log('新代码')  // [!code ++]

  console.log('错误代码')  // [!code error]
  console.log('警告代码')  // [!code warning]
}
```

## 📚 常见问题

::: details 如何使用 {v-pre}？
在代码块中添加 `{v-pre}` 可以防止 VitePress 编译 Vue 语法：

```html {v-pre}
<template>
  <div>{{ message }}</div>
</template>
```
:::

::: details 如何添加图片？
```markdown
![图片描述](./path/to/image.png)
```

图片路径可以是：
- 相对路径：`./images/pic.png`
- 绝对路径：`/public/images/pic.png`
- 外部链接：`https://example.com/pic.png`
:::

::: details 如何创建锚点链接？
```markdown
[跳转到某个标题](#标题名称)
```

注意：
- 标题名称需要转换为小写
- 空格用 `-` 替换
- 特殊符号需要移除
:::

## 🎓 写作建议

1. **使用清晰的标题层级**
   - 一级标题：文章主标题
   - 二级标题：章节标题
   - 三级标题：小节标题

2. **合理使用容器**
   - `tip`：给出建议和技巧
   - `warning`：提醒注意事项
   - `danger`：重要警告
   - `details`：补充详细信息

3. **代码块最佳实践**
   - 指定语言以获得语法高亮
   - 使用行高亮突出重点
   - 使用代码组展示多种方案

4. **图文并茂**
   - 适当添加图片说明
   - 使用表格整理数据
   - 用列表归纳要点
