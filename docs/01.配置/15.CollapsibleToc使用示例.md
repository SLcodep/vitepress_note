# CollapsibleToc 组件使用示例

<CollapsibleToc 
  title="快速导航" 
  :defaultOpen="true" 
  :minLevel="2" 
  :maxLevel="4" 
/>

## 基本使用

直接在 Markdown 文件中使用 `<CollapsibleToc />` 即可自动生成目录。

### 默认配置

- 目录默认是**折叠**状态
- 自动提取 **h2-h3** 级别的标题
- 支持点击跳转到对应位置

## 自定义配置

### 设置默认展开

```vue
<CollapsibleToc :defaultOpen="true" />
```

### 自定义目录标题

```vue
<CollapsibleToc title="文章导航" />
```

### 提取更多级别的标题

```vue
<!-- 提取 h2-h4 -->
<CollapsibleToc :minLevel="2" :maxLevel="4" />
```

### 组合使用

```vue
<CollapsibleToc 
  title="快速导航" 
  :defaultOpen="true" 
  :minLevel="2" 
  :maxLevel="4" 
/>
```

## 功能特性

### 自动提取标题

组件会自动扫描页面中的标题元素，无需手动维护目录列表。

### 平滑滚动

点击目录项时，页面会平滑滚动到对应位置，提升用户体验。

### 响应式设计

组件支持深色/浅色模式，并在移动端有良好的显示效果。

### 层级缩进

不同级别的标题会有不同的缩进，清晰展示文档结构。

## 样式定制

组件使用 VitePress 的 CSS 变量，会自动适配您的主题样式：

- `--vp-c-brand-1`: 主题色
- `--vp-c-bg-soft`: 背景色
- `--vp-c-divider`: 分割线颜色
- `--vp-c-text-1/2/3`: 文字颜色

## 注意事项

1. 确保文章中的标题有唯一的 ID（VitePress 会自动生成）
2. 组件需要在文档内容加载后才能提取标题
3. 建议将组件放在文章标题下方，正文内容之前
